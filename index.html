Why it's different

Built for beauty,
not just functionality

Every output looks like it was designed in Keynote, not generated by code.

✨

GSAP animations

Industry-standard animation engine with Apple's expo-out easing curve. Headlines drop in, bullets stagger, quote watermarks scale up behind text.

🎨

Apple design system

SF Pro typeface, exact Apple color tokens, Liquid Glass-inspired fills, and the same typography scale used across iOS and macOS.

📺

Video & YouTube embeds

Drop any YouTube URL into a media slide. The skill auto-converts it to an embed. Also supports local video files and images.

📱

Fully responsive

Maintains 16:9 aspect ratio at any window size. All spacing uses clamp() and percentage-based values so nothing ever breaks.

🌗

Dark and light themes

Dark by default. Set "theme": "light" in your spec for a clean, bright Apple-style look. Both themes use the same token system.

⌨️

Keyboard navigation

Arrow keys, spacebar, swipe on mobile, click left/right half of slide. Press F to go fullscreen. Works in any browser.

📦

Zero dependencies

Each deck is a single self-contained HTML file. Share it, email it, or host it anywhere — no build tools, no Node, no servers.

✍️

Apple Style Guide writing

Sentence case headings, active voice, no filler words, serial comma. Claude writes copy that matches Apple's style guide by default.

Slide types

Six layouts, infinite combinations

Claude picks the right layout for each piece of content automatically.

title

Opening slide or section break. Centered headline with optional eyebrow label, subheading, and icon.

content

The workhorse. Headline with bullet points that stagger in one by one. Optional footnote.

quote

Full-bleed quote with a decorative watermark " behind the text, eyebrow label, and attribution.

two-column

Side-by-side comparison cards. Each column has a title and bullet list. Columns animate in from opposite sides.

media

YouTube embed, video file, or image. Auto-detects the media type from the URL. 16:9 frame with caption.

closing

Strong final statement with a call to action and contact info. Headline scales in with spring easing.
